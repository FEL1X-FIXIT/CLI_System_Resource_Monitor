# CLI_System_Resource_Monitor
A basic, colorfull system resource monitor for terminal. Mainly build it for monitoring my system when running heavy inference loads or any resource hungry scripts/apps etc.

#CPU:
\n-CPU Model
\n-Multi CPU Support
\n-Core & Threads Count
\n-Clock Speed/Max Clock Speed
\n-Current Clock Speed
\n-Current Temp
\n-Current Load %

#RAM:
-RAM Per Slot
-Type (eg.DDR4)
-Size (eg.16GB)
-CFG Speed, Max and Current Setting
-ECC (Simply Shown as -ECC)
-Total RAM & Current Usage

#GPU:
-Multi GPU Support
-GPU Name/Product Details and Current Slot
-Usage %
-VRAM (Total and Used)
-Core Temp
-Power Consumption (in Watts)
-Current Clock Speed

#DRIVES:
-HDD, SSD, NVME Etc.
-Supports Multiple Drives
-Live In/Out Traffic in MB/s
